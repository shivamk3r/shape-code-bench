# Freshness Check

Check whether the current change requires source-of-truth documentation updates.

Review:

- `README.md`
- `docs/benchmark-spec.md`
- `AGENTS.md`

Answer:

- Does the public overview still match the implementation?
- Does the benchmark spec still match the code and tests?
- Should agent instructions or shared asset layout be updated?
- Are any generated artifacts, run outputs, or planning files being mistaken for
  live instructions?
